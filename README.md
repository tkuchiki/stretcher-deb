# stretcher-deb
stretcher debian package

## Download deb package

https://github.com/tkuchiki/stretcher-deb/releases

## Installation

```
git clone https://github.com/tkuchiki/stretcher-deb
cd stretcher-deb
```

### Setup

Install dependencies

```
sudo make setup
```

### Build deb package

```
make
```

### Install

```
sudo make install
```

## Operability confirmed

- debian8
- ubuntu14.04
- ubuntu16.04
