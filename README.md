# stretcher-deb
stretcher debian package

## Download

https://github.com/tkuchiki/stretcher-deb/releases

## Installation

```
git clone https://github.com/tkuchiki/stretcher-deb
cd stretcher-deb
```

### Setup

Install dependencies

```
sudo make setup
```

### Build deb package

```
make
```

### Install

```
sudo make install
```
